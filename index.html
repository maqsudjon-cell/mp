01 / ABOUT

Turning curiosity into tools.

I design, build and ship small useful things end to end — usually a single self-contained file deployed on GitHub Pages. If it helps one student study, it was worth building.

Most of what I make is for IELTS students: a free ecosystem of listening and reading tests, an essay platform with AI feedback, a speaking lab, and a 40-day challenge tracker. Alongside that I make tiny learning apps and release music as Matsumi.

I'm self-taught across web, Python, electronics and sound, and I speak Uzbek, English (C1) and some Korean. Curious about a lot — still figuring out where it all points.
