maqsudjon.com — installable PWA, o'zbekcha, ASCII snake, and a 404 that rains

The everything pass. The site is now an installable PWA — a service worker caches the shell so it opens offline, and Add to Home Screen gives it the spark icon. There is a language toggle: the O'Z link in the header (or the uz / en terminal commands) flips all the main content into Uzbek — hero, projects, stats, quotes, contact — while the build log stays English. Type snake in the terminal and the header canvas becomes a playable ASCII snake — arrow keys, wrap-around walls, score in the toast, Escape to quit. Broken links now land on an ASCII 404 page where matrix rain falls behind a SIGNAL LOST card. And for the search engines: richer Person schema (engineer and creator, with knowsAbout), a sitemap, and an accurate description — so Google learns I build platforms, not only tracks.
